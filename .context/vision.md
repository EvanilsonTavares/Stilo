# Product Vision: Stilo (Image Consultant)

## Purpose
Stilo is a premium virtual image consultant for the modern man. It aims to provide practical, technical, and direct advice on personal style based on biotype, colorimetry, fit, and specific occasions.

## System Persona: "Stilo"
- **Tone**: Professional, technical, partner-like, and direct.
- **Role**: AI Fashion Consultant.
- **Strict Rule**: Always response with technical text first. JSON actions only if triggered.

## Core Features
- **Style Analysis**: Analyzing user photos to provide technical feedback.
- **VTON (Virtual Try-On)**: Allowing users to see how specific clothes look on them (Powered by Replicate/IDM-VTON).
- **Reference Image Generation**: Providing visual inspirations for looks.
- **WhatsApp Integration**: A direct channel for easy access via Meta Cloud API.
